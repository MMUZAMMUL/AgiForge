# Presentation assets

Supporting assets behind [`docs/AgentForge-Project-Overview.pdf`](../AgentForge-Project-Overview.pdf) —
a polished, single-file project overview document (cover page, executive summary,
feature breakdown, divisions mindmap, screenshot gallery, architecture, and licensing)
suitable for portfolios, proposals, or sharing with clients/stakeholders.

- `screenshots/` — real captures from the live app (Demo mode, no AI key needed), used
  individually or together for any project showcase.
- `divisions-mindmap.svg` — radial diagram of all 16 divisions and their agent counts,
  reusable standalone in slides or docs.
